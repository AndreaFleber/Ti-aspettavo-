# Ti Aspettavo!

> Il vuoto chiama. L’opera nasce solo quando qualcuno entra con intenzione.

## Concept

Questa non è un’opera statica.  
Non c’è forma né messaggio imposto.  
Non ci sono indicazioni da seguire.

L’opera esiste **solo quando qualcuno sceglie di entrarvi**.  
Ogni emozione, pensiero, dubbio o resistenza che accende in chi entra **trasforma il vuoto in opera**.  
Anche l’indifferenza o il disprezzo diventano reazione e, quindi, parte dell’esperienza.

## Come Interagire

1. Apri il file `ti-aspettavo.html`.  
2. Osserva lo spazio vuoto e il titolo senza aspettative.  
3. Clicca o interagisci: l’opera si attiva.  
4. Lascia che il vuoto diventi esperienza personale e condivisa.  

## Filosofia

L’artista non completa l’opera: crea le condizioni.  
Chi entra decide se, come e quando il vuoto diventa gesto.  
Il titolo **“Ti Aspettavo!”** è l’invito a partecipare e l’eco della responsabilità condivisa tra opera e spettatore.

## Note Tecniche (opzionale)

- Il file `ti-aspettavo.html` è completamente minimale e funzionante su qualsiasi browser moderno.  
- Il repository può essere ampliato con media, note o versioni digitali interattive.
